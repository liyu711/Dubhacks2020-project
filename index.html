# Changelog
All the notable changes for this project are documented in this file.
The github for the project: https://github.com/liyu711/Dubhacks2020-project.git
We mainly focused on the core functionality and UI. By tracking the process, we were able to mark down things and achieve our goal for buiding a website.

### Core Functionality
- input the Mouse DPI, FOV, and Sensibility for different games and the web calculator is able to give an output of the Mouse Sensibility
- have a well-track Changelog
- have a credit page for the web developers
- the web calculator can also refer the users to the game pages.

### User Interface
- ask for the input
- easy to find the content and follow the Instruction
- have the dropdown menu so that users can easily find what they are looking for
- the search engine
- a functionaing web calculator

Introduction & Instruction

Mouse Sensitivity Calculator allows users to calculate the mouse sensitivity for different games.
We are currently supporting three games:
1) Tom Clancy’s Rainbow Six Siege, 2) Valorant, 3) Counter-Strike: Global Offensive.
